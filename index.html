Ns-3-Power-Control

NS3 Network Simulator with power control of WiFi transmissions. Implementation of different proposed mechanisms for joint power and rate control.

Download .zip Download .tar.gz View on GitHub

This project contains the implementations of three different power and rate control algorithms: PARF[1], APARF[2] and Minstrel-Piano[3] for the NS3 simulator. There is also included tests where this algorithms can be compared in different scenarios.
