Coordinated Management of Rate, Power and Carrier Sense Threshold for IEEE 802.11 Networks

NS3 Network Simulator with power control of WiFi transmissions. Implementation of different proposed mechanisms for joint rate, power and carrier sense threshold control.

Download .zip Download .tar.gz View on GitHub

This project contains the NS3 Simulator and the implementations of different rate, power and carrier sense threshold control algorithms: PARF, APARF, Minstrel-Piano, RRPAA and PRCS for the NS3 simulator. There is also included tests where this algorithms can be compared in different scenarios.

Coordinated Management of Rate, Power and Carrier Sense Threshold for IEEE 802.11 Networks is maintained by mrichart
